## Running a gateway

To run a gateway: 

- Checkout the [repository](https://github.com/unruggable-labs/evmgateway-v2/tree/main/v2).

- Run the following setup commands
    - `foundryup`
    - `forge install`
    - `bun i`


- Copy the `.env.example` template to `.env` and define the appropriate environment variables.
    - `INFURA_KEY`
    - `ALCHEMY_KEY`
