## Running a gateway

If you would like to run your own gateway you are more than welcome to do so.

To run a gateway: 

### Checkout

- Checkout the [repository](https://github.com/unruggable-labs/evmgateway-v2/tree/main/v2).

`git clone https://github.com/unruggable-labs/evmgateway-v2.git`

### Setup

- Run the following setup commands
    - `foundryup`
    - `forge install`
    - `bun i`

### Configuration

- Copy the `.env.example` template to `.env` and define the appropriate environment variables.
    - `INFURA_KEY`
    - `ALCHEMY_KEY`

### Run it

- Run the gateway using the following commands:

`bun run serve <chain> [port]`

**Supported chains:** `base` `op` `arb1` `scroll` `taiko` `utaiko`

**Default port:** `8000`
